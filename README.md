# Royal Canin - Where to buy - html build

## Requirements
Node v12

## Setup
```
npm install
```
```
npm install --global gulp-cli
```
```
npm install --save-dev gulp
```

## To run local development
```
gulp
```

## To run build
```
gulp build
```
